# Commands
- whoami -> tells the user name
- touch → create a new file
- ls -> list out files/directories
- vi -> open the editor to edit the file
  Press I -> enter the insert mode
  press ESC to exit the insert mode
  :wq or :exit or :x -> save and exit
  :w -> save your file
  :q -> quit the editor
  :q! - quit w/o saving
- cat -> see the content of file
- head → display first 10 lines
- tail → display last 10 lines
- clear -> clear the screen

Folder is called directory in linux.

- mkdir -> make a directory
- cd -> change directory

- rm -> remove the file
- rmdir -> remove a directory
- mv -> rename the file
- rm -r -> remove forcefully
